# 摩托车零部件采购管理系统 —— 功能列表

> 系统定位：纯内部进销存（**采购入库 + 领用出库**，无对外销售）。
> 核心规则：**库存的任何增减，都必须由管理员审核通过后才触发**（职责分离）。
> 配套文档：[CLAUDE.md](../CLAUDE.md)、[数据库表结构](database_schema.sql)、[ER 图](database_er.drawio)、[用例图](usecase_diagram.puml)。

---

## 一、角色与权限总览

| 角色 | 定位 | 能做 | 不能做 |
|---|---|---|---|
| **采购员**（purchase） | 只管进货 | 建档（供应商/零件）、开采购单、录物流、提交入库、标记付款 | 无权入库、无权领用、不能直接改库存 |
| **领用人员**（requisition） | 车间/生产领料 | 开领用单、提交出库申请 | 不能直接扣库存、不参与采购 |
| **管理员**（admin） | 审核中枢 | 审核档案/入库/出库、库存增减、用户管理、参数配置、报表 | —— |

## 二、功能模块清单

### 1. 登录与认证
- 1.1 用户登录（账号 + 密码）
- 1.2 验证密码（登录的必含步骤）
- 1.3 登录失败提示（显示登录错误）
- 1.4 退出登录
- 1.5 按角色加载对应菜单与操作权限

### 2. 供应商管理（采购员建档 / 管理员审核）
- 2.1 新建供应商资料（采购员提交）
- 2.2 编辑 / 查看供应商信息（编码、名称、联系人、电话、地址、信用评级）
- 2.3 供应商列表查询、分页、按状态/信用评级筛选
- 2.4 审核供应商资料（管理员：通过 / 驳回）
- 2.5 终止 / 启用合作状态
- 2.6 逻辑删除供应商

### 3. 零部件管理（采购员建档 / 管理员审核锁价）
- 3.1 新建零部件资料（采购员提交）
- 3.2 编辑 / 查看零件信息（编码、名称、型号、规格、单位、分类、所属供应商）
- 3.3 零件列表查询、分页、按分类/供应商筛选
- 3.4 审核零件档案、**锁定采购单价**（管理员）
- 3.5 设置库存预警值
- 3.6 逻辑删除零件

### 4. 采购管理（采购入库流程）
- 4.1 创建采购单（选零件、填采购数量）
- 4.2 **自动计算订单总价**（数量 × 单价，明细小计自动生成）
- 4.3 采购单明细管理（增删改一张单里的多条零件明细）
- 4.4 录入物流信息（物流公司、运单号、发货时间）
- 4.5 标记供应商发货、登记预计/实际到货日期
- 4.6 货到后**提交入库申请**
- 4.7 对账后**标记已付款**
- 4.8 取消采购单
- 4.9 采购单列表查询（按状态：采购中 / 待入库审核 / 已入库 / 已取消）

### 5. 领用管理（领用出库流程）
- 5.1 创建领用单（选零件、填领用数量、填用途/部门）
- 5.2 领用单明细管理（一张单多条零件明细）
- 5.3 **提交出库申请**
- 5.4 取消领用单
- 5.5 领用单列表查询（按状态：待审核 / 已出库 / 已取消）

### 6. 审核中心（管理员专属，库存变动闸门）
- 6.1 审核**采购入库**申请 → 通过后**库存增加**、更新最近入库时间
- 6.2 审核**领用出库**申请 → **先验证库存充足** → 通过后**库存减少**、更新最近出库时间
- 6.3 审核驳回（退回申请人并附原因）
- 6.4 审核任务/待办列表（待我审核的单据）

### 7. 库存管理
- 7.1 库存查询（当前数量、安全库存、仓库位置、最近出入库时间）
- 7.2 库存明细按零件/仓位筛选
- 7.3 **库存预警**（低于预警值/安全库存高亮提醒）
- 7.4 库存预警参数配置（管理员设置预警值/安全库存）
- 7.5 库存只读保护：禁止手工直接改数字，只能经审核出入库变更

### 8. 用户管理（管理员专属）
- 8.1 新建 / 编辑 / 查看用户账号
- 8.2 分配角色（采购员 / 领用人员 / 管理员）
- 8.3 启用 / 禁用账号
- 8.4 重置密码
- 8.5 用户列表查询、按角色/状态筛选

### 9. 统计与报表
- 9.1 采购统计（按日期统计订单数、总金额、均价）
- 9.2 库存预警汇总（严重不足 / 不足 / 充足）
- 9.3 采购明细全信息查询（含零件、供应商）
- 9.4 领用出库统计（按部门/用途/时间）
