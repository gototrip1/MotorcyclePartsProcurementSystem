# 数据来源说明

## 概述

本系统的数据库设计和模拟数据参考了以下来源：

## 1. 供应商数据参考

供应商名称参考了以下真实摩托车配件品牌和供应商：

### 国际知名品牌：
- **博世 (Bosch)** - 全球领先的汽车零部件供应商
- **电装 (DENSO)** - 日本顶级汽车零部件制造商
- **布雷博 (Brembo)** - 意大利知名制动系统制造商
- **昭和 (Showa)** - 日本知名减震器和悬架系统制造商
- **KYB** - 日本知名的液压设备制造商，产品包括减震器
- **日清纺 (Nisshinbo)** - 日本汽车制动系统供应商
- **NGK** - 日本知名火花塞制造商
- **盖茨 (Gates)** - 全球知名的传动系统和皮带制造商

### 国内供应商：
参考了国内主要摩托车配件批发市场的供应商结构

## 2. 产品/零部件数据参考

零部件分类和命名参考了：
- 摩托车通用配件分类标准
- 阿里巴巴1688工业品采购平台的配件分类
- 京东企业购的摩托车配件分类
- 摩托车维修店的实际配件清单

### 主要分类：
1. **发动机类**：活塞环、缸体、曲轴、凸轮轴等
2. **车架类**：减震器、方向把、主支架、链轮等
3. **电气类**：磁电机、点火线圈、火花塞、大灯等
4. **制动类**：刹车片、刹车蹄块、刹车泵等
5. **传动类**：传动皮带、离合器、普利珠等
6. **外观件**：挡泥板、后视镜、座垫等

## 3. 价格数据参考

价格范围参考了：
- 阿里巴巴1688批发平台的价格区间
- 京东商城摩托车配件的零售价格
- 实体摩托车配件批发市场的报价单

### 价格区间：
- 普通零件：10-500元
- 核心部件（发动机类、传动类）：500-5000元
- 零售价通常是采购价的1.2-1.5倍

## 4. 数据库表结构参考

表结构设计参考了：
- 常见ERP系统的数据库设计
- 阿里巴巴、京东等电商平台的后台数据结构
- 制造业进销存管理系统的标准设计

### 设计原则：
1. 遵循数据库设计范式
2. 适当的数据冗余以提高查询效率
3. 逻辑删除代替物理删除，保护数据安全
4. 统一的时间戳字段便于数据追踪

## 5. 业务逻辑参考

业务场景和数据关系参考了：
- 中小型摩托车配件贸易公司的实际业务流程
- 采购管理系统的标准功能模块
- 库存管理系统的最佳实践

## 声明

本系统中的模拟数据仅用于开发和测试目的，所有供应商名称、产品信息等均为虚构，如有雷同纯属巧合。如需在生产环境中使用，请替换为真实的业务数据。
